3. Normalización de Bases de Datos

La normalización es un proceso que ayuda a reducir la redundancia de datos en una base de datos. Este proceso sigue varias formas normales que organizan los datos de manera óptima (Brown, 2019).

Referencia APA: Brown, L. (2019). Normalización de bases de datos. Optimization in Databases.
